Assign Block

This type of block is used to declare variables, and do some operations.

Rules

Every single variable and operation must be separated with space (' ')

There always must be a single equals-sign ('=')

Left side of equals-sign must be a single variable

Right side of equals-sign must contain only variables (already declared) and operations (+ - * /)

Write Block

This type of block is used print data to output

Rules

Everything inside this block get's printed to the screen

To print out a variable, it must be declared and no other data than the variable itself must be inside this block
