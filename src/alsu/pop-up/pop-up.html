This type of block is used to declare variables, and do some operations.

Rules

- Every single variable and operation must be separated with spaces (' ')

- There always must be a single equals-sign ('=')

- On left side of equals-sign must be a single variable

- The right side of equals-sign must contain only variables (declared) and operations (+ - * / %)

This type of block is used print data to output

Rules

- Everything inside this block will be displayed on output screen

- To display a variable value:

-- must be declared

-- no other text than the variable itself must be inside this block

{{ block[1].data }}
